# 项目名称

欢迎访问 [tfseek.top](https://tfseek.top)！这是一个基于 Vue 和 Element Plus 的浏览器起始页项目。

## 项目描述

此项目旨在为用户提供一个简洁、美观的浏览器起始页，方便快捷地访问常用网站和服务。

## 使用技术

- **Vue**: 用于构建用户界面
- **Element Plus**: 组件库，提供丰富的 UI 组件

## 安装

```bash
npm install
```

## 运行项目

```bash
npm run dev
```
